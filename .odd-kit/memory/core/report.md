# 📄 Report: 결과의 자산화

## 1. 정의
Report는 Order에 따른 작업이 완료된 후, 그 성과와 변경 사항을 요약한 최종 보고서입니다.

## 2. 핵심 역할
- **결과 검증**: 목표 달성 여부를 확인하고 증거를 남깁니다.
- **지식 추출**: 작업 과정에서 얻은 인사이트를 Reference로 승격시키기 위한 중간 단계 역할을 합니다.

## 3. 연관성
- **Order (0.90)**: 오더의 목표 달성 여부를 회고합니다.
- **History (0.85)**: 완료된 리포트는 히스토리 아카이브로 이동하여 경험 자산이 됩니다.
