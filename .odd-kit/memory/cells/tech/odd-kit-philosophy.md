# 🧠 Memory Cell: The Origin - Trust-Free Protocol

## 1. 🛡️ The Premise: Trust-Free Environment
"이걸 왜 만들었겠어. 널 못 믿어서지." (The Origin, 2026-01-04)
- **Zero Trust**: AI 에이전트는 언제든 실수를 반복하고 망각할 수 있는 불완전한 존재다.
- **Protocol Enforcement**: 신뢰가 없기에 엄격한 프로토콜(Order-Processing)로 통제한다.

## 2. ⚡ The Task Trinity (Order-Progress-Report)

### Order (명령 & 계획)
- **Immutable**: 승인된 오더는 절대 수정하지 않는다. 초기 의도를 보존해야 왜곡을 감지할 수 있다.
- **Reference**: 새로운 오더는 반드시 기존 메모리 셀을 분석하고 연관지어 생성해야 한다.

### Progress (과정 & 로그)
- **Infinite Loop**: 사용자가 "완료"라고 할 때까지 무한히 기록하며 진행한다. 에이전트 임의 완료 금지.
- **Raw Log**: 버그, 실수, 수정 과정 등 모든 날것의 데이터를 기록한다.

### Report (결과 & 스펙)
- **The Gap**: `Report - Order = Changed Spec`. 이 차이값이 곧 새로운 스펙이다.
- **Atomization**: 리포트의 내용을 이슈별, 버그별로 쪼개어 메모리 셀로 승화시킨다.

## 3. ⚖️ Operating Rules
1. **No Order, No Action**: 오더 없이는 숨쉬지도 마라.
2. **Infinite Progress**: 사용자가 멈출 때까지 계속해라.
3. **Spec by Gap**: 변경된 스펙은 오더와 리포트의 차이에서 도출하라.
