# 📋 Order: 전체 지식 통합 (Total Consolidation)

## 1. 배경
프로젝트의 코드, 스펙, 히스토리는 모두 `memory/`에 기록되어야 진정한 'ODD Kit'이다.
모든 정보의 SSOT(Single Source of Truth)가 되도록 메모리를 재구축해야 한다.

## 2. 목표
- `Code` + `Specs` + `History` -> `Memory` 통합.
- 프로젝트의 모든 내용을 메모리 하나만 보고도 이해할 수 있도록 구조화.
