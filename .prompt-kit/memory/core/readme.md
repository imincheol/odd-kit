# 📖 README: 시스템 사용 설명서

## 1. 정의
프로젝트의 루트 및 `.odd/` 디렉토리에 위치하며, 인간 개발자와 AI 모두를 위한 구조 가이드입니다.

## 2. 핵심 역할
- **구조 설명**: 복잡하게 쪼개진 원자적 레퍼런스들의 지도를 제공합니다.
- **온보딩**: 새로운 개발자나 AI가 시스템을 이해하는 첫 화면 역할을 합니다.

## 3. 연관성
- **ODD Prompt (0.80)**: 개발 실무는 ODD Prompt가, 구조 안내는 Readme가 담당합니다.
- **Reference (0.75)**: 레퍼런스 라이브러리의 전체 목록을 안내할 수 있습니다.
