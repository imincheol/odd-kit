# ⚡ Active: 현재의 초점

## 1. 정의
Active는 현재 진행 중인 모든 프로세스(Order, Progress)가 머무르는 공간입니다. 시스템의 '활성 메모리' 역할을 수행합니다.

## 2. 핵심 역할
- **작업 가시성**: 현재 무엇이 수행되고 있는지 한눈에 파악하게 합니다.
- **컨텍스트 집중**: 작업 중인 문서들만을 분리하여 AI의 주의력 분산을 막습니다.

## 3. 연관성
- **Task (0.90)**: 실행 중인 태스크 파일들의 물리적 거주지입니다.
- **Roadmap (0.75)**: 로드맵의 항목이 실행 상태가 되면 Active로 전환됩니다.
