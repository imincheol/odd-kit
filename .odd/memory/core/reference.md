# 📚 Reference: 지식의 원천

## 1. 정의
Reference는 시스템 운영에 필요한 가이드라인, 환경 설정, 도메인 지식 등을 담고 있는 정적인 지식 창고입니다.

## 2. 핵심 역할
- **장기 기억**: 세션이 바뀌어도 변하지 않는 규칙과 지식을 제공합니다.
- **동적 연관성**: 문헌 간의 연관성(Relatedness)을 통해 지식의 연결망을 형성합니다.

## 3. 연관성
- **Relatedness Guideline (1.0)**: 레퍼런스 시스템 자체를 규정하는 가이드입니다.
- **ODD Prompt (0.90)**: 프롬프트가 레퍼런스를 참조하여 작업을 수행합니다.
