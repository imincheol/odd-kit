# 🤖 Prompt: AI와의 인터페이스

## 1. 정의
Prompt는 AI 에이전트에게 전달되는 지시문 세트입니다. 시스템 프롬프트(System Prompt)와 작업별 프롬프트 등이 포함됩니다.

## 2. 핵심 역할
- **역할 정의**: AI가 Antigravity로서 어떻게 행동해야 하는지 규정합니다.
- **컨텍스트 주입**: Reference와 Active의 정보를 Prompt에 결합하여 실행 명령을 만듭니다.

## 3. 연관성
- **ODD Prompt (0.95)**: 프로젝트의 메인 프롬프트와 직접 연결됩니다.
- **Reference (0.85)**: 프롬프트 구성의 재료가 되는 지식입니다.
