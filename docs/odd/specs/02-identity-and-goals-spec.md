# Spec: Project Identity & Core Values (v2.0.0)

> **"Curing AI Amnesia with Order-Driven Context Management."**

## 🎯 1. Mission & Vision
ODD Kit은 AI 에이전트가 긴 작업 과정에서 맥락을 유실하는 '망각(Amnesia)' 현상을 치료하는 것을 목표로 합니다. 대화 세션에 의존하지 않고, 로컬 저장소의 파일 시스템을 '지식의 저장소'로 활용하여 어떤 환경에서도 즉각적인 자아 복원(Self-Restoration)이 가능하게 합니다.

## 🌟 2. Core Values (Key Features)

### 2.1 기억의 원자화 (Atomization)
거대한 문서를 뉴런과 같이 작고 명확한 단위(Memory Cell)로 쪼개어 참조의 정확도를 높입니다. 필요한 지식만 핀셋으로 집어내듯 호출할 수 있습니다.

### 2.2 지식의 선순환 (Turn-Based Cycle)
지식을 먼저 전수조사하여 동기화하고(Turn-Memory), 그 기반 위에서 작업을 실행(Turn-Order)하는 순환 구조를 가져 지식의 파편화를 방지합니다.

### 2.3 무결성 유지 (Pillar Sync)
Specs, Template, Engine 세 기둥이 항상 동일한 지식을 공유하도록 강제합니다. 하나가 변하면 나머지도 함께 변하는 동기화 메커니즘을 가집니다.

### 2.4 명시적 참조 (Explicit-Based)
모든 행동의 근거가 되는 메모리 셀을 오더와 리포트에 명시하여 AI의 사고 능력을 투명하게 추적하고 가이드합니다.

## ⚖️ 3. Comparison with AI Frameworks
ODD Kit은 단독 사용뿐만 아니라 다른 AI 도구들과 병행하여 시너지를 낼 수 있습니다. 자세한 내용은 `AI-FRAMEWORKS.md` 문서를 참조하십시오.

- **vs Spec Kit (Google)**: Spec Kit은 설계를 돕고, ODD Kit은 그 설계가 실행되고 기억되는 과정을 관리합니다.
- **vs Claude Skills**: ODD Kit의 운영 프로토콜을 Skill로 등록하여 자동화된 컨텍스트 관리를 수행할 수 있습니다.
