# UI/UX 가이드라인

이 문서는 화면의 레이아웃, 컴포넌트 배치, 인터랙션 상태를 정의합니다.

## 📱 Layout Strategy

- **Mobile First**: 기본 모바일 레이아웃 구조.
- **Desktop Breakpoint**: 데스크탑 확장 시 변경되는 레이아웃 (예: 사이드바 노출).

## 🧩 Common Components

- **Buttons**: Primary, Secondary, Disabled 상태 정의.
- **Inputs**: Default, Focus, Error, Success 상태 정의.
- **Modals**: 팝업의 위치(Center/Bottom) 및 오버레이 동작.
