# Origin & References (불변의 원본)

이 폴더는 프로젝트의 **변하지 않는 기준(Immutable Context)**과 **외부 참조 문서(External References)**를 보관합니다.

## 🚫 Read-Only Policy

이 폴더 내의 문서들은 프로젝트 진행 도중 임의로 수정되어서는 안 됩니다.
Order 수행 시 이 문서들은 **참조(Reference)**의 용도로만 사용하십시오.
